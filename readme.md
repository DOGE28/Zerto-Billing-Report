# Zerto Billing Report
A very simple script that scrapes the following information from each Zorg in the Boise and St George ZVMs:

* Zorg Name
* Total VMs
* Total vCPU
* Total Provisioned Memory
* Total Provisioned Storage
* Total Used Storage
